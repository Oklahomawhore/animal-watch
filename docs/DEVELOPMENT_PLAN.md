## Phase 2: 算法开发

### 2.1 动物检测算法

#### 目标
- 检测视频画面中的林麝
- 统计数量
- 追踪运动轨迹

#### 技术方案
```
方案A: YOLOv8 + 自定义训练
- 优点: 速度快，部署简单
- 缺点: 需要标注数据

方案B: 传统CV + 深度学习混合
- 背景建模 + 目标检测
- 适合固定摄像头场景
```

#### 开发步骤
1. [ ] 获取测试视频/图片
2. [ ] 数据标注（100-500张）
3. [ ] 模型训练
4. [ ] 部署测试
5. [ ] 性能优化

### 2.2 草量分析算法

#### 目标
- 分析圈舍草量是否充足
- 预警草料不足

#### 技术方案
```
方案: 图像分割 + 像素统计
- 分割出草地区域
- 计算绿色像素比例
- 设定阈值预警
```

#### 开发步骤
1. [ ] 收集不同草量的样本图片
2. [ ] 标注草地区域
3. [ ] 训练分割模型
4. [ ] 设定预警阈值
5. [ ] 集成告警系统

### 2.3 算法服务架构

```
┌─────────────────┐
│  算法服务 (Python)  │
│  - YOLO检测        │
│  - 草量分析        │

         │
    ┌────┴────┐
    ▼         ▼
┌─────────┐ ┌─────────┐
│  抓拍图片  │ │  实时视频  │
└─────────┘ └─────────┘
```
