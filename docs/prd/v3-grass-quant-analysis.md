## 二、剩料量化算法方案

### 方案A：空/满比对法（推荐MVP）

**技术栈**:
```python
# 核心算法：像素差异比对
import cv2
import numpy as np

def calculate_grass_coverage(current_img, empty_baseline, full_baseline):
    """
    计算食槽草量覆盖率
    
    参数:
        current_img: 当前帧食槽区域 (ROI)
        empty_baseline: 空槽基准图
        full_baseline: 满槽基准图
    
    返回:
        coverage_ratio: 0-100% 草量覆盖率
    """
    # 1. 预处理：转HSV，提取绿色/枝叶特征
    curr_hsv = cv2.cvtColor(current_img, cv2.COLOR_BGR2HSV)
    empty_hsv = cv2.cvtColor(empty_baseline, cv2.COLOR_BGR2HSV)
    
    # 2. 绿色区域检测 (草/枝叶)
    # HSV绿色范围
    lower_green = np.array([35, 40, 40])
    upper_green = np.array([85, 255, 255])
    
    curr_green = cv2.inRange(curr_hsv, lower_green, upper_green)
    empty_green = cv2.inRange(empty_hsv, lower_green, upper_green)
    
    # 3. 计算差异
    # 当前绿色像素占比 - 空槽绿色像素占比
    curr_ratio = np.sum(curr_green > 0) / curr_green.size
    empty_ratio = np.sum(empty_green > 0) / empty_green.size
    
    # 4. 归一化到0-100%
    # 假设满槽时 green_ratio = full_max
    full_max = 0.6  # 经验值，可配置
    empty_min = empty_ratio
    
    coverage = (curr_ratio - empty_min) / (full_max - empty_min) * 100
    return max(0, min(100, coverage))

# 事件检测
def detect_feeding_events(coverage_history, timestamps):
    """
    从覆盖率历史检测进食事件
    """
    events = []
    
    # 检测首次降至50%
    for i, cov in enumerate(coverage_history):
        if cov < 50 and (i == 0 or coverage_history[i-1] >= 50):
            events.append({
                'type': 'half_empty',
                'timestamp': timestamps[i],
                'coverage': cov
            })
    
    # 检测降至0%
    for i, cov in enumerate(coverage_history):
        if cov < 10 and (i == 0 or coverage_history[i-1] >= 10):
            events.append({
                'type': 'empty',
                'timestamp': timestamps[i],
                'coverage': cov
            })
    
    return events
```

**实现步骤**:
1. **初始化配置**: 管理员后台框选食槽ROI区域
2. **基准图采集**: 空槽图(投料前) + 满槽图(刚投完料)
3. **定时采样**: 每10分钟提取一帧分析
4. **变化检测**: 与基准图比对计算覆盖率
5. **事件触发**: 基于规则生成预警

### 方案B：深度学习分割（二期）

**技术栈**:
```python
# 使用轻量级分割模型
import torch
from torchvision import models

# 方案：MobileNetV3 + DeepLabV3
# 或使用YOLOv8-seg

class GrassSegmentor:
    def __init__(self, model_path):
        # 加载预训练模型
        self.model = torch.load(model_path)
        
    def segment(self, image):
        """分割草料区域"""
        # 输出：草料mask
        mask = self.model(image)
        coverage = np.sum(mask) / mask.size * 100
        return coverage, mask
```

**标注需求**:
- 每张图标注食槽区域+草料像素
- 建议：500-1000张训练图
- 工具：LabelMe / CVAT

## 五、关键问题解答

### Q1: 是否需要RTSP视频流？
**答**: 剩料量化不需要连续视频流，定时截图足够。建议:
- 分析用: 每10分钟截图一帧
- 查看用: 按需拉流预览

### Q2: 海康云能否满足需求？
**答**: 海康云主要提供存储和基础AI，剩料量化需自定义算法。建议:
- 云端存储视频备查
- 边缘端跑自定义算法

### Q3: 成本最低的方案？
**答**: 
- 硬件: 瑞芯微RK3588盒子 (¥800)
- 带宽: 仅上报结果 (¥50/月)
- 云端: 轻量服务器 (¥100/月)
- **单养殖场总计: ¥950/月**

### Q4: 与现有ISAPI方案冲突吗？
**答**: 不冲突，可以共存:
- ISAPI事件: 运动检测，低功耗
- RTSP截图: 剩料分析，按需拉流
