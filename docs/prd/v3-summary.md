## 💡 算法方案A（推荐）

**核心**: 空槽/满槽基准图比对

```python
# 简化版流程
1. 管理员框选食槽ROI区域
2. 上传空槽图 + 满槽图（基准）
3. 每10分钟截图，与基准图比对
4. 计算绿色像素占比变化
5. 生成覆盖率趋势曲线 (0-100%)
```

**优势**:
- ✅ 无需AI训练，成本低
- ✅ 4周可跑通MVP
- ✅ 精度满足"何时吃完"需求

## ⚡ 关键决策点

### 1. 用海康云还是自建？
**建议**: 海康云做存储备查，**边缘盒子跑自定义算法**
- 海康云不提供"剩料量化"这种细分功能
- 边缘盒子更灵活，成本低

### 2. 需要RTSP视频流吗？
**不需要连续流**，定时截图足够：
- 分析：每10分钟1帧
- 查看：按需拉流

### 3. 最低成本多少？
**单养殖场**: 
- RK3588盒子: ¥800 (一次性)
- 月运营: ¥150 (带宽+云端存储)
